Hohmann Orbit Transfer Simulation (GMAT)

This project demonstrates a **Hohmann transfer** between two circular Earth orbits using the General Mission Analysis Tool (GMAT). A Hohmann transfer is an efficient orbital maneuver used to move a spacecraft from one circular orbit to another using two engine burns.

Objective

To simulate and visualize a two-impulse transfer maneuver where a spacecraft transfers from a lower circular orbit to a higher circular orbit around Earth.

Method

The simulation is performed in GMAT using a two-body gravitational model. The spacecraft initially begins in a circular low Earth orbit. A transfer burn is applied at periapsis to place the spacecraft onto an elliptical transfer orbit. A second burn is applied at apoapsis to circularize the orbit at the target altitude

Concepts Demonstrated

* Orbital mechanics
* Hohmann transfer trajectory
* Delta-V maneuver planning
* Orbit propagation

Tools Used

* GMAT (General Mission Analysis Tool)

Outcome

The simulation shows the spacecraft leaving its initial orbit, entering an elliptical transfer orbit, and finally circularizing at the target orbit. The trajectory and orbital parameters are visualized using GMAT’s orbit visualization tools.
